### рЯЖШ Rescue Connect вАФ Unified Emergency Response Platform
> *Best Women in Tech вАФ Prayatna 3.0 (36-hour National Hackathon) | 1650+ participants across India*

Navigation isn't the problem вАФ **the absence of seamless coordination is.** RescueConnect reimagines emergency response as a single intelligent ecosystem connecting the person reporting, the authority orchestrating, and the vehicle executing вАФ all in real time.

**3-Layer Architecture:**
- рЯУ± **Layer 1 вАФ Detection & Reporting:** React interface for SOS/social signal ingestion вЖТ Supabase (PostgreSQL)
- рЯІ† **Layer 2 вАФ Intelligence & Decision:** YOLO + Tesseract OCR ¬Ј spaCy NER ¬Ј Gemini Vision / Groq classification ¬Ј OSM Nominatim geolocation
- рЯЪЧ **Layer 3 вАФ Routing & Orchestration:** SUMO + TraCI ¬Ј OSRM ¬Ј LSTM + GCN ¬Ј XGBoost ¬Ј DQN-based rerouting ¬Ј WebSockets (2 Hz) syncing Flutter apps & dashboards вАФ built for Indian conditions (festivals, monsoons, dense traffic)

**Tech:** `React` `Vite` `TailwindCSS` `FastAPI` `Supabase` `Gemini VLM` `Groq` `spaCy` `YOLO` `Tesseract` `OSRM` `SUMO` `TraCI` `LSTM+GCN` `XGBoost` `DQN` `WebSockets` `Flutter` `Google Cloud`
